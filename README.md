# Heart Disease Prediction - Machine Learning Project

## Project Overview
This project involves a comparative analysis of various machine learning classification models to predict the risk of heart disease. Based on clinical medical data, we evaluate and compare the performance of different algorithms, specifically focusing on Random Forest and Support Vector Machines (SVM), to identify the most accurate predictive model.
